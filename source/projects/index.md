项目页面用于记录大创、科研训练、课程实践、编程项目和自主探索。每个项目建议单独写成一篇文章，并放入“项目”分类。

## 项目记录模板

每个项目可以包含以下部分：

- 项目名称；
- 项目背景；
- 我的工作；
- 技术路线；
- 项目进度；
- 项目成果；
- 相关图片；
- 相关 GitHub 仓库链接。

## 新建项目文章

```bash
npx hexo new project "项目名称"
```

生成后可以在文章中继续补充实验图片、流程图、代码片段和仓库链接。
